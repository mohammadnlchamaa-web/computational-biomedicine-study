Machine Learning-Based Biomedical Cancer Classification Using Feature Reduction and Ensemble Models

Computational Biomedicine Research Study

Author: Mohammad Chamaa

Computational Biomedicine Study Laboratory

Dataset: Breast Cancer Wisconsin (Diagnostic)

Abstract

Breast cancer diagnosis remains a critical challenge in modern healthcare, where early and accurate detection significantly improves patient outcomes and survival rates.

In this study, machine learning techniques were applied for automated breast cancer classification using the Breast Cancer Wisconsin (Diagnostic) dataset.

Principal Component Analysis (PCA) was used to reduce dimensionality while preserving the majority of dataset variance.

Logistic Regression, Random Forest, and Support Vector Machine (SVM) models were implemented and evaluated using accuracy, precision, recall, F1-score, and ROC-AUC.

Results demonstrated strong predictive performance across all models, with SVM achieving the highest overall performance.
